    \subsubsection{Storing cleaned
dataframes}\label{storing-cleaned-dataframes}

    In the beginning we had three seperate datasets. Due to tidiness issues,
we have merged the \texttt{rt} dataframe to \texttt{tw} dataframe as
each type of observational unit forms a table.

    I did not merge \texttt{im} dataframe to \texttt{tw} dataframe because
each dataset holds different types of observational units for each tweet
of WeRateDogs Twitter account. In \texttt{tw} dataframe, we hold every
statistical and informative data of a tweet such as the favourite count,
tweet text, name and breed of the dog mentioned in the tweet. On the
other hand the \texttt{im} dataframe contains the dog breed predictions
of each dog mentioned in each WeRateDogs Twitter account by using
neutral network, which is totally a different type of observation.
